Simulation Hybride & IA

EIMLIA-3M-TEU

[cite_start]

Étude des Impacts Médico-Financiers, Logistiques de l'IA avec 3 Modèles Triant à l’Entrée des Urgences [cite: 1]

Description du Projet

[cite_start]Le projet EIMLIA-3M-TEU vise à évaluer, par une simulation prospective validée rétrospectivement, les performances multidimensionnelles de trois architectures d'IA (NLP, LLM, JEPA) pour le triage aux urgences[cite: 5, 6]. L'étude se distingue par une approche intégrée combinant critères cliniques, organisationnels et économiques sur une cohorte de 600 000 patients du CHU de Lille.

Objectif Principal

Identifier l'architecture optimale pour réduire de 25% les temps d'attente et de 20% les hospitalisations inappropriées.

Méthodologie

[cite_start]

Simulation multimodale hybride : Simulation à Événements Discrets (DES) [cite: 10] + [cite_start]Systèmes Multi-Agents (SMA) [cite: 7, 8] [cite_start]enrichie de Jumeaux Numériques[cite: 15].

Innovation

[cite_start]

Comparaison inédite d'un modèle JEPA (séries temporelles + texte) face aux LLM génératifs et NLP classiques[cite: 18].

Données

[cite_start]

Entraînement et validation sur 6 ans de données historiques (2018-2023) anonymisées, valorisées selon les tarifs ATIH[cite: 17].

Architectures IA Évaluées

1. Modèle NLP

    [cite_start]
  • Tech : Doc2Vec [cite: 19] + Réseau Neuronal Profond.
  • Force : Extraction rapide de symptômes critiques, latence très faible (120ms).
  • Limite : Compréhension contextuelle limitée des cas complexes.

2. Modèle LLM

    [cite_start]
  • Tech : FlauBERT Fine-tuné [cite: 23] + XGBoost Hybride.
    • [cite_start]
  • Force : Raisonnement clinique contextuel, explicabilité via SHAP[cite: 25].
  • Sécurité : Module anti-hallucinations basé sur le référentiel FRENCH.

3. Modèle JEPA

  • Tech : Joint-Embedding Predictive Architecture.
    • [cite_start]
  • Force : Encodage conjoint texte/séries temporelles pour prédire la dégradation[cite: 18].
  • Perf : Sensibilité >96% sur les urgences vitales.

Infrastructure de Simulation

La validation écologique est assurée par une stack technique avancée permettant des stress-tests jusqu'à 200% de charge. [cite_start]Le Process Mining [cite: 9] permet une génération réaliste des parcours.

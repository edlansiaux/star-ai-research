Simulation Hybride & IA

EIMLIA-3M-TEU

Étude des Impacts Médico-Financiers, Logistiques de l'IA avec 3 Modèles Triant à l’Entrée des Urgences

Description du Projet

Le projet EIMLIA-3M-TEU vise à évaluer, par une simulation prospective validée rétrospectivement, les performances multidimensionnelles de trois architectures d'IA (NLP, LLM, JEPA) pour le triage aux urgences. [cite_start]L'étude se distingue par une approche intégrée combinant critères cliniques, organisationnels et économiques sur une cohorte de 600 000 patients du CHU de Lille[cite: 6, 13].

Objectif Principal

[cite_start]

Identifier l'architecture optimale pour réduire de 25% les temps d'attente et de 20% les hospitalisations inappropriées[cite: 16].

Méthodologie

[cite_start]

Simulation multimodale hybride : Simulation à Événements Discrets (DES) + Systèmes Multi-Agents (SMA) enrichie de Jumeaux Numériques[cite: 12].

Innovation

[cite_start]

Comparaison inédite d'un modèle JEPA (séries temporelles + texte) face aux LLM génératifs et NLP classiques[cite: 8].

Données

[cite_start]

Entraînement et validation sur 6 ans de données historiques (2018-2023) anonymisées[cite: 13].

Architectures IA Évaluées

1. Modèle NLP

    [cite_start]
  • Tech : Doc2Vec + Réseau Neuronal Profond[cite: 245].
  • Force : Extraction rapide de symptômes critiques, latence très faible (120ms).
    • [cite_start]
  • Limite : Compréhension contextuelle limitée des cas complexes[cite: 278].

2. Modèle LLM

    [cite_start]
  • Tech : FlauBERT Fine-tuné + XGBoost Hybride[cite: 279].
    • [cite_start]
  • Force : Raisonnement clinique contextuel, explicabilité via SHAP[cite: 303].
  • Sécurité : Module anti-hallucinations basé sur le référentiel FRENCH.

3. Modèle JEPA

    [cite_start]
  • Tech : Joint-Embedding Predictive Architecture (LeCun)[cite: 42].
    • [cite_start]
  • Force : Encodage conjoint texte/séries temporelles pour prédire la dégradation[cite: 320].
    • [cite_start]
  • Perf : Sensibilité >96% sur les urgences vitales (niveaux 1-2)[cite: 360].

Infrastructure de Simulation

[cite_start]La validation écologique est assurée par une stack technique avancée permettant des stress-tests jusqu'à 200% de charge (scénario de crise)[cite: 14].

    [cite_start]
  • Simulation : Couplage Rockwell Arena (DES) et AnyLogic 8.9 (SMA)[cite: 146].
    • [cite_start]
  • Calcul : Cluster GPU NVIDIA A100 avec orchestration Kubernetes pour l'inférence IA[cite: 372].
    • [cite_start]
  • Process Mining : Utilisation de Celonis pour la génération réaliste des parcours patients[cite: 203].
